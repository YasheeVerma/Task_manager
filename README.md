#  Task Manager App

Task Manager application that allows users to create, update, delete, and manage tasks efficiently.

---

##  Features

*  Add new tasks
*  View all tasks
*  Mark tasks as completed
*  Delete tasks
*  Basic validation

---

##  Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

### Frontend

* HTML
* CSS
* JavaScript

##  How to Run Locally

### 1. Clone the repository

```
git clone https://github.com/YasheeVerma/Task_manager.git
cd Task_manager
```

### 2. Install dependencies

```
npm install
```

### 3. Start the server

```
node server.js
```

### 4. Run frontend

Open:

```
frontend/index.html
```

in your browser
